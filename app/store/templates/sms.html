📱 Números Android

Configura múltiples números Android para recibir SMS directamente desde tus celulares usando SMS Forwarder

Cargando números Android...

📋 Instrucciones para SMS Forwarder:
  1. Agrega el número Android: Ingresa el número con código de país (ej: +573001234567) y guarda
  2. Copia la URL del webhook: Haz clic en "Editar" del número Android y copia la URL del webhook que aparece
  3. Abre SMS Forwarder en tu celular:
    • Ve a "Webhooks" o "HTTP POST"
    • Crea un nuevo webhook
    • Pega la URL copiada en el campo "URL"
    • Método: POST
    • Content-Type: application/json
  4. Configura el Body JSON: En el campo "Body" o "JSON Body", pega exactamente esto (reemplaza TU_API_KEY con tu API key si la configuraste): 
    {"from_number": "{in-number}", "to_number": "{out-number}", "message_body": "{msg}", "api_key": "TU_API_KEY"}
    Nota: Si no configuraste API key, puedes omitir el campo "api_key" o dejarlo vacío: 
    {"from_number": "{in-number}", "to_number": "{out-number}", "message_body": "{msg}"}
  5. Guarda y activa: Guarda la configuración y activa el webhook en SMS Forwarder

💡 Tip: Los placeholders {in-number}, {out-number} y {msg} son reemplazados automáticamente por SMS Forwarder con los datos reales del SMS recibido.
